Smart Lock Installation Process

1

Assess Door & Compatibility

We verify your door has the correct bore hole size (2-1/8″), backset measurement, and door thickness for your chosen model. We also check WiFi signal strength at the door.

2

Remove Existing Hardware

Existing deadbolt or lock removed. Door prep verified and modified if needed for perfect fit of the new smart lock hardware.

3

Install & Align Lock

Smart lock installed with careful alignment — the bolt must throw cleanly and the strike plate must engage fully. We check door gap and adjust strike plate as needed.

4

App Setup & Configuration

We help you download the app, connect to WiFi or your smart home hub, create access codes, set up auto-lock timers, and integrate with Alexa/Google Home if desired.
