Door Hardware for Access Control

Access control systems require compatible door hardware. Common configurations:

  • Electric Strike: Replaces the standard strike plate. Released by an electrical signal. Works with existing locksets. Most common for retrofits.
  • Electromagnetic Lock (Maglock): 600–1,200 lb holding force. Installed at the top of the door frame. Fail-safe (releases on power loss) — requires door position sensor for fire code compliance.
  • Electric Mortise Lock: Full-function lock with motorized throw bolt. Best for high-security applications. More expensive but most secure.
