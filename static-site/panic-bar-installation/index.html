When Is a Panic Bar Required?

🔴 IBC Code Requirement

The International Building Code (IBC), adopted throughout Texas, requires panic hardware (exit devices) on doors serving occupancies of 50 or more people in specific use groups.

Required in:

  • Assembly occupancies (A-1 through A-5): theaters, restaurants, churches
  • Education occupancies (E): schools, daycares
  • High-hazard occupancies (H)
  • Some Mercantile (M) and Business (B) occupancies over 50 persons

🔦 Fire Marshal Inspection

The Dallas Fire Marshal inspects commercial occupancies for IBC and NFPA 101 Life Safety Code compliance. A missing or malfunctioning exit device can result in:

  • Failed fire inspection
  • Fines and citations
  • Mandatory closure until corrected
  • Liability in emergency events

We install UL-listed devices and can provide documentation for your inspection records.

Panic Bar Configurations

  • Rim Exit Device: Simplest installation — mounted on door surface with rim strike on frame. Best for most standard commercial doors.
  • Mortise Exit Device: Full mortise lock body with panic bar actuating the deadbolt and latch. Most secure option for high-security exit doors.
  • Surface Vertical Rod (SVR): Rods extend to top and bottom of door frame. Required for double doors where a surface-mounted rim device isn't practical.
  • Concealed Vertical Rod (CVR): Rods run inside the door — clean exterior appearance. Requires special door preparation.

Exterior Trim Options

Panic hardware is push-to-exit from inside. For controlled entry from outside:

  • Key cylinder trim: Standard key cylinder allows entry from outside
  • Lever trim: Lever handle for free public access
  • Access control trim: Card reader or keypad connected to the latch
  • Blank/exit only: No outside entry — alarm-connected versions available
